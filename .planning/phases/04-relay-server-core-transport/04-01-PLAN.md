    **Step 5: Create echo WebSocket server** (`apps/feishu/src/echo-server.ts`).

    Export a `startEchoServer(port: number)` function that:
    - Creates a `ws.WebSocketServer` on the given port
    - On each connection, listens for text messages
    - Parses incoming text as JSON
    - If valid JSON, sends the same JSON string back
    - If invalid JSON, sends `{ error: "invalid JSON" }` back
    - Returns `{ server, close: () => void }` for clean shutdown

    When run directly (not imported), start on port 9099:
    ```typescript
    const isMain = process.argv[1]?.endsWith("echo-server.ts") || process.argv[1]?.endsWith("echo-server.js");

      const { server } = startEchoServer(9099);
      console.log("Echo server listening on ws://localhost:9099");
    }
    ```
