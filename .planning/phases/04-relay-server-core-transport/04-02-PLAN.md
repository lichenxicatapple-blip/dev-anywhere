    ```typescript
      registerProxy(proxyId: string, ws: WebSocket): void;

      getProxy(proxyId: string): WebSocket | undefined;
      listProxies(): string[];

      getClientsForProxy(proxyId: string): WebSocket[];
      getBoundProxy(clientWs: WebSocket): string | undefined;
    }
    ```

    `registerProxy`: sets `proxyId -> ws` in the map. If a proxy with the same proxyId already exists, close the old connection and replace.

    `bindClient`: checks proxyId exists in proxies map first. Returns false if not found. Otherwise sets binding.
    `getClientsForProxy`: iterates clientBindings, collects all WebSocket keys where value === proxyId.

    **Step 1: Create router module** (`apps/relay/src/router.ts`).

    Per D-07/D-08/D-10:

    ```typescript
    import { MessageEnvelopeSchema, RelayControlSchema } from "@cc-anywhere/shared";
    import type { WebSocket } from "ws";
    import type { Logger } from "pino";
    import type { RelayRegistry } from "./registry.js";

    // 尝试解析消息：先试 RelayControl，再试 MessageEnvelope
    export type ParseResult =
      | { kind: "control"; message: RelayControlMessage }
      | { kind: "envelope"; message: MessageEnvelope; raw: string }
      | { kind: "invalid"; error: string };

    export function parseMessage(data: string): ParseResult;

    // 将 proxy 发来的 MessageEnvelope 转发给绑定到该 proxyId 的所有 client
    export function routeProxyMessage(
      raw: string,
      proxyId: string,
      registry: RelayRegistry,
      logger: Logger,
    ): void;

    // 将 client 发来的 MessageEnvelope 转发给该 client 绑定的 proxy
    export function routeClientMessage(
      raw: string,
      clientWs: WebSocket,
      registry: RelayRegistry,
      logger: Logger,
    ): void;
    ```

    `parseMessage`: Try `JSON.parse(data)` first. Then `RelayControlSchema.safeParse()`. If matches, return `{ kind: "control" }`. Otherwise `MessageEnvelopeSchema.safeParse()`. If matches, return `{ kind: "envelope", raw: data }`. Otherwise `{ kind: "invalid" }`.

    `routeProxyMessage`: Parse with `parseMessage`. If `kind === "envelope"`, iterate `registry.getClientsForProxy(proxyId)`, for each client with `readyState === WebSocket.OPEN`, call `clientWs.send(raw)`. Forward the original raw string (no re-serialization). If `kind === "invalid"`, log warning with proxyId and error. Control messages from proxy should not reach this function (handled in handler).

    `routeClientMessage`: Parse with `parseMessage`. If `kind === "envelope"`, get `proxyId = registry.getBoundProxy(clientWs)`. If no binding, send relay_error to client. If proxyId bound but proxy `readyState !== WebSocket.OPEN`, send relay_error "proxy_offline". Otherwise `proxyWs.send(raw)`. If `kind === "invalid"`, log warning.

    When sending relay_error, serialize: `JSON.stringify({ type: "relay_error", code: "...", message: "..." })`.

    **Step 2: Create proxy handler** (`apps/relay/src/handlers/proxy.ts`).

    Per D-05/D-09/D-10:

    ```typescript
    import type { WebSocket, WebSocketServer } from "ws";
    import type { Logger } from "pino";
    import type { RelayRegistry } from "../registry.js";
    import { parseMessage, routeProxyMessage } from "../router.js";

    export function handleProxyConnection(
      ws: WebSocket,
      registry: RelayRegistry,
      logger: Logger,
    ): void;
    ```

    On connection:
    1. Set `(ws as any).isAlive = true`
    2. Listen for `pong` event: set `isAlive = true`
    3. Listen for `message` event (data as string):
       - Call `parseMessage(data.toString())`
       - If `kind === "control"` and `type === "proxy_register"`: call `registry.registerProxy(msg.proxyId, ws)`. Log registration. Store proxyId on the ws instance: `(ws as any).proxyId = msg.proxyId`.
       - If `kind === "envelope"`: call `routeProxyMessage(data.toString(), (ws as any).proxyId, registry, logger)`. If proxyId not yet registered, send relay_error.
       - If `kind === "invalid"`: log warning, ignore.
    4. Listen for `close` event: call `registry.unregisterProxy((ws as any).proxyId)` if proxyId exists. Log disconnection.
    5. Listen for `error` event: log error.

    Export a `setupProxyHeartbeat(wss: WebSocketServer, interval: number): NodeJS.Timeout` function:
    - Every `interval` ms, iterate `wss.clients`
    - If `isAlive === false`, call `ws.terminate()`
    - Otherwise set `isAlive = false`, call `ws.ping()`
    - Default interval: 30000ms (30 seconds)
