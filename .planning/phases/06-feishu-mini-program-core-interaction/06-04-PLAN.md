1. Define blacklist (D-29):
```typescript
const WATCH_BLACKLIST = new Set([
  "node_modules", ".git", "dist", "build", ".next", ".nuxt",
  "__pycache__", ".venv", ".tox", "target", ".gradle",
]);
```

2. Implement `FileWatcher` class:
```typescript
export class FileWatcher {
  private watcher: fs.FSWatcher | null = null;
  private throttleTimers = new Map<string, NodeJS.Timeout>();
  private readonly workDir: string;
  private readonly onUpdate: (dirPath: string, entries: DirEntry[]) => void;
  private readonly throttleMs: number;

  constructor(workDir: string, onUpdate: (dirPath: string, entries: DirEntry[]) => void, throttleMs = 2000);
```

3. `start()` method:
   - Use `fs.watch(this.workDir, { recursive: true })` to watch the entire tree
   - In the change handler, extract the directory from the changed filename
   - Check if any path segment is in WATCH_BLACKLIST -- if so, ignore
   - Throttle: if a timer exists for this directory, skip. Otherwise, set a 2s timer that, when fired, calls `listDirectory(fullDirPath)` and passes result to `onUpdate`

4. `getInitialTree(depth = 2): Map<string, DirEntry[]>`:
   - BFS/DFS traversal of workDir up to `depth` levels
   - Skip blacklisted directories
   - Return map of path -> entries

5. `stop()`: close watcher, clear all throttle timers
