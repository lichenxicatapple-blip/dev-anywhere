**Step 1: Rewrite `apps/proxy/src/index.ts`**

Replace the Phase 1 type-check placeholder with the actual CLI entry point. The file should be minimal -- all logic lives in PtyManager.

```typescript
import { PtyManager } from "./pty-manager.js";
import { createNoopTap } from "./tap.js";

// per D-02: 所有命令行参数直接透传给 claude
const claudeArgs = process.argv.slice(2);

const manager = new PtyManager({
  claudeArgs,
  tap: createNoopTap(),
  stdin: process.stdin,
  stdout: process.stdout,
});

manager.start();
```

This is the complete file. No commander, no argument parsing, no configuration -- per D-02 and D-03, all args pass through to claude and cc-anywhere's own config comes from environment variables (not needed in Phase 2).
