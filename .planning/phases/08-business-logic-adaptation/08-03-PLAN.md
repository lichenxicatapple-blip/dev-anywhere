1. Create `apps/web/src/hooks/use-relay-setup.ts` (per D-15, D-18):

```typescript
import { useEffect, useRef } from 'react';

```

The hook `useRelaySetup()` does the following in a single `useEffect([], ...)`:

a. Resolve relay URL (D-18 priority chain):
   ```typescript
   const stored = localStorage.getItem('cc_relayUrl');
   const envUrl = import.meta.env.VITE_RELAY_URL as string | undefined;
   const relayUrl = stored || envUrl || window.location.origin;
   useAppStore.getState().setRelayUrl(relayUrl);
   ```

b. Create WebSocketManager singleton (stored in useRef):
   ```typescript
   const ws = new WebSocketManager();
   wsRef.current = ws;
   ```

c. Create RelayClient using clientId from app-store:
   ```typescript
   const clientId = useAppStore.getState().clientId;
   const relay = new RelayClient(ws, clientId);
   relayRef.current = relay;
   ```

d. Create timers ref:
   ```typescript
   const timers: Timers = { reconnect: null, coldStartDone: false };
   timersRef.current = timers;
   ```

e. Wire WebSocket status changes to phase-machine:
   ```typescript
   const unsubStatus = ws.onStatusChange((connected) => {
     handleWsStatusChange(connected, timersRef.current!, relayRef.current!);
   });
   ```

f. Wire relay messages to phase-machine:
   ```typescript
   const unsubRelay = relay.onMessage((msg) => {
     void handleRelayMessage(msg as Record<string, unknown>, timersRef.current!, relayRef.current!);
   });
   ```

g. Set up visibilitychange listener (per D-08):
   ```typescript
   const handleVisibility = () => {
     if (document.visibilityState === 'visible' && wsRef.current && !wsRef.current.isConnected()) {
       wsRef.current.connect(relayUrl);
     }
   };
   document.addEventListener('visibilitychange', handleVisibility);
   ```

h. Connect WebSocket:
   ```typescript
   ws.connect(relayUrl);
   ```

i. Cleanup function returns:
   ```typescript
   return () => {
     unsubStatus();
     unsubRelay();
     document.removeEventListener('visibilitychange', handleVisibility);
     ws.close();
   };
   ```

Export the WebSocketManager ref so pty-test and future pages can access it:
```typescript
export let wsManagerRef: WebSocketManager | null = null;
```
Set `wsManagerRef = ws` during setup and `wsManagerRef = null` during cleanup.

Also export `relayClientRef` with the same pattern for pages that need relay-client access.
